# Функциональные требования

## FR-001 Регистрация пользователя
Система должна позволять пользователю зарегистрироваться по email и паролю.

## FR-002 Авторизация
Система должна позволять пользователю входить в личный кабинет.

## FR-003 Просмотр каталога товаров
Система должна отображать список товаров по категориям.

## FR-004 Поиск товаров
Система должна предоставлять поиск по названию товара.

## FR-005 Фильтрация товаров
Система должна позволять фильтровать товары по:

* категории
* цене
* бренду

## FR-006 Карточка товара
Система должна отображать:

* название
* описание
* цену
* рейтинг
* наличие

## FR-007 Корзина
Система должна позволять добавлять товары в корзину.

## FR-008 Оформление заказа
Система должна позволять оформить заказ.

## FR-009 Онлайн-оплата
Система должна позволять оплатить заказ банковской картой.

## FR-010 История заказов
Система должна отображать историю покупок пользователя.

## FR-011 Избранное
Система должна позволять сохранять товары в избранное.

## FR-012 Управление товарами
Администратор должен иметь возможность:

* добавлять товары
* изменять товары
* удалять товары

## FR-013 Управление заказами
Администратор должен иметь возможность изменять статус заказа.

## FR-014 Отчетность
Система должна предоставлять данные по продажам и заказам.
